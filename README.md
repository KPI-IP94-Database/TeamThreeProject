# TeamThreeProject
The organization consists of subgroup of students of the IP-94 group,
KPI, ICT faculty.

The project is about to hold data of Ukrainian entrants, such as
results of the national evaluating (ZNO), high school priorities, etc.

## Technology Stack

* Client: Node.JS
* DB: SQLite
